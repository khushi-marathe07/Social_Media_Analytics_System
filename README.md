# Social_Media_Analytics_System

## 📌 Project Overview
The Social Media Analytics System is a database project developed using Oracle SQL. It is designed to manage and analyze social media data, including users, posts, comments, likes, and followers. The project demonstrates database design, data management, and SQL query execution.

## 🛠️ Technologies Used
- Oracle Database
- SQL
- SQL Plus

## ✨ Features
- User Management
- Post Management
- Comment Management
- Like Management
- Follower Management
- Data Analysis using SQL Queries

## 🗂️ Database Tables
- Users
- Posts
- Comments
- Likes
- Followers

## 📚 SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT
- UPDATE
- DELETE
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)

## 🎯 Project Objective
To design a relational database for a social media platform and perform data analysis using SQL queries.

## 🚀 How to Run
1. Open Oracle SQL Plus.
2. Create the required tables.
3. Insert the sample data.
4. Execute the SQL queries to analyze the data.

## 📖 Learning Outcomes
- Database Design
- SQL Query Writing
- Data Retrieval
- Data Analysis
- Relational Database Management
